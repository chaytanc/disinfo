# Platform Usage Guidelines

## Shared Infrastructure

This project uses **shared infrastructure** to enable immediate access while maintaining system integrity and supporting ongoing development.

## Usage Terms

By using this repository, you agree to:

### ✅ Permitted Use
- **Research and academic purposes** 
- **Personal analysis projects**
- **Educational and learning purposes**
- **Contributing improvements back to the project**
- **Reasonable usage** (not exceeding ~1000 API calls/day per user)

### ❌ Prohibited Use  
- **Commercial applications** without permission
- **High-volume automated usage** that impacts system performance
- **Reselling or redistributing** the service
- **Creating competing hosted services**
- **Removing attribution** to this project
- **Bypassing authentication** for unauthorized access
- ❌ **Sharing outside of your organization without the creator's permission**

## For Heavy Usage

If your research requires:
- **>1000 API calls per day**
- **Commercial deployment**
- **Custom infrastructure setup**
- **White-label usage**

Please **contact the maintainer** to discuss:
- Custom setup assistance
- Dedicated resources
- Commercial licensing
- Collaboration opportunities

## Technical Implementation

The shared infrastructure enables:
- **Immediate setup** - no configuration needed
- **Centralized management** for reliability
- **Usage analytics** for project improvement
- **Community features** and collaboration

## Community Expectations

We trust users to use this resource responsibly:

1. **Give credit** - cite this [project](https://arxiv.org/abs/2507.20066) in publications/presentations
2. **Contribute back** - share improvements, bug fixes, new features using Pull Requests 
3. **Report issues** - help improve the platform for everyone using Github Issues
4. **Respect limits** - don't abuse the shared infrastructure
5. **Support sustainability** - consider supporting development if this tool is valuable to your work
